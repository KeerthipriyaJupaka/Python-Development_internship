# Task 5 - Currency Converter

## Description
A Python-based currency converter that uses an Exchange Rate API to convert one currency into another in real time.

## Features
- Real-time currency conversion
- API integration
- User input for currencies and amount
- Error handling for invalid input
- User-friendly interface

## Technologies Used
- Python 3
- Requests Library
- Exchange Rate API

## How to Run

```bash
python Task5.py
```

## Install Required Library

```bash
python -m pip install requests
```
