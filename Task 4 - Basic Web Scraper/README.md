# Task 4 - Basic Web Scraper

## Description
A Python-based web scraper that extracts top news headlines from a website using Requests and BeautifulSoup libraries.

## Features
- Fetch webpage content
- Extract headlines from HTML
- Display top news headlines
- Exception handling for network errors
- User-friendly interface

## Technologies Used
- Python 3
- Requests Library
- BeautifulSoup Library

## How to Run

```bash
python Task4.py
```

## Install Required Libraries

```bash
python -m pip install requests beautifulsoup4
```
