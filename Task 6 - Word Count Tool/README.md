# Task 6 - Word Count Tool

## Description
A Python-based Word Count Tool that analyzes a text file and displays the number of lines, words, characters, and the most frequently used words.

## Features
- Count total lines
- Count total words
- Count total characters
- Display most common words
- Exception handling for file errors
- User-friendly interface

## Technologies Used
- Python 3
- Collections Module

## How to Run

```bash
python Task6.py
```
