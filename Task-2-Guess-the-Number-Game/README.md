# Task 2 - Guess the Number Game

## Description
A Python-based number guessing game where the program generates a random number and the user tries to guess it.

## Features
- Random number generation
- Higher/Lower hints
- Attempt counter
- User-friendly interface
- Error handling

## Technologies Used
- Python 3
- Random Module

## How to Run

```bash
python Task2.py
```
