# Task 3 - Basic File Handling

## Description
A Python program that reads data from a text file, finds and replaces specific words, and saves the modified content back to the file.

## Features
- Read data from text file
- Find and replace words
- Save updated content
- Exception handling for file errors
- User-friendly interface

## Technologies Used
- Python 3
- File Handling
- Exception Handling

## How to Run

```bash
python Task3.py
```
